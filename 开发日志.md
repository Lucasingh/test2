# 开发日志

## 2026-07-05 DeepSeek AI 自动审核接入

### 功能
- 在"审核文献"页面新增"🤖 AI 一键审核"按钮
- 点击后调用 DeepSeek API，对数据库中所有未审核文献自动分类和打标签
- 分批处理（每批 10 篇），避免单次请求过长

### 新增/修改文件
| 文件 | 内容 |
|------|------|
| `ai_classifier.py` | DeepSeek 分类逻辑（prompt + API 调用 + JSON 解析） |
| `database.py` | 新增 `batch_ai_classify()` 方法（按标题匹配写入结果） |
| `app.py` | 导入 ai_classifier，审核页面加 AI 按钮，改名"审核文献" |

### 测试结果
- 13 篇文献，6 秒完成分类
- 高置信 12 篇（92%），中等 1 篇，无低置信
- 分类准确率 100%

### UI 变更
- "✅ 人工审核" → "📋 审核文献"
