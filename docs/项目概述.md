4. **置信度计算**: 实际得分 / 最大可能得分

### 3.6 任务管理 - tasks.py

**双模式支持**:

| 模式 | 适用场景 | 特点 |
|------|---------|------|
| Redis + RQ | 生产环境 | 持久化、分布式、可监控 |

**内存模式关键设计**:

```python
class RedisTaskManager:
    def __init__(self):
        self._use_redis = False
        self.tasks = {}
        self._worker_thread = None
        self._lock = threading.Lock()
        
        if CONFIG.get('redis', {}).get('enabled', False):
            # 尝试连接 Redis...
            pass
    
    def _run_memory_worker(self):
        """后台线程循环执行待处理任务"""
        while True:
            pending_task = None
            with self._lock:
                for task_id, task in self.tasks.items():
                    if task['status'] == 'pending':
                        pending_task = task
                        break
            
            if pending_task:
                # 执行任务...
                result = crawl_task(params)
                with self._lock:
                    self.tasks[task_id]['status'] = 'completed'
            
            time.sleep(1)
```

## 十、已知问题与解决方案

### 10.1 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 数据库连接失败 | MySQL 服务未启动或密码错误 | 检查 MySQL 服务，修改 `config.py` 中的密码 |
| 采集无数据 | 网络问题或数据源API限制 | 检查网络连接，部分数据源需要特殊权限 |
| Redis 连接警告 | Redis 未启用或未安装 | 在配置中设置 `enabled: false` |
| 页面白屏 | CSS `:has()` 选择器匹配过宽 | 使用精确选择器 `[data-testid="stTextInput"]:has(input)` |
| 自动登录失效 | iframe 安全限制 | 使用 `window.parent.localStorage` 和 `execCommand` |

### 10.2 Streamlit 开发注意事项

| 规则 | 说明 |
|------|------|
| `set_page_config()` 必须是第一个 Streamlit 命令 | 否则报错 `StreamlitSetPageConfigMustBeFirstCommandError` |
| 组件 key 不能与 session_state 变量同名 | 否则报错 `StreamlitAPIException` |
| `display: none` 的元素无法获得焦点 | 使用 `position: fixed; left: -9999px` 移出屏幕 |
| `st.components.v1.html` 在 iframe 中执行 | 需通过 `window.parent` 访问主页面 |
